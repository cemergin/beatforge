# SWANA & Indian Drum Pattern Library
## Playable 16th-Note Grid Patterns for Web Drum Machine

### How to Read This Document

Each position in the grid is one **16th note**. A measure of 4/4 = 16 positions.
Odd meters are handled by adjusting the total grid length (e.g., 9/8 = 18 sixteenth notes).

**Traditional Hand Drum Voices:**
- `D` = Doum (bass, center strike) — low, resonant
- `T` = Tek (edge strike) — bright, high pitch
- `K` = Ka (muted slap) — ghost note, dead stroke

**Drum Kit Mapping:**
- `KK` = Kick drum (maps from Doum)
- `SN` = Snare / rimshot (maps from Tek)
- `HH` = Hi-hat (maps from Ka / fills / ghost notes)

**Indian Tabla Voices (adapted to grid):**
- `D` = Ge/Ghe (bass, bayan) — left hand, low
- `T` = Na/Ta/Tin (treble, dayan) — right hand, high
- `K` = Dhin/Dha (combined stroke) — both hands together

**Symbols:**
- `x` = strike
- `.` = rest / silence
- `|` = bar line (visual grouping only)
- Pipe groups show metric subdivisions where relevant

# PART 3: INDIAN RHYTHMS
# =============================================

## NOTATION KEY FOR INDIAN PATTERNS

Indian rhythms (tala) use a different conceptual framework:

- **Sam** (X) = Beat 1, the point of resolution. Always emphasized.
- **Khali** (0) = "Empty" beat — played with a wave instead of a clap. Marks the midpoint contrast.
- **Tali** (+) = Clap positions marking section starts.
- **Vibhag** = Section groupings within the tala.
- **Matra** = Individual beat units.

**Tabla Bols (syllables) mapped to drum grid:**
- `Dha` / `Ghe` = Both hands (bass + treble) → D line (doum equivalent)
- `Na` / `Ta` / `Tin` = Right hand treble only → T line (tek equivalent)
- `Dhin` = Combined resonant stroke → K line (combined)
- `Tun` / `Ti` = Light stroke → ghost note

For the 16-step grid adaptation, each Indian beat (matra) typically occupies
2 or 4 grid positions depending on the tala's subdivision requirements.
