# Comprehensive Drum Pattern Library
## For Web Drum Machine Step Sequencer

**Notation Key:**
- `x` = hit
- `X` = accent (hard hit)
- `.` = rest (silence)
- `o` = open hi-hat
- `g` = ghost note (soft hit)
- `|` = barline
- Each character = one 16th note (unless noted otherwise)
- Steps 1, 5, 9, 13 = beats 1, 2, 3, 4 in 4/4 time

**Instruments:**
- `HH` = closed hi-hat
- `OH` = open hi-hat
- `RD` = ride cymbal
- `RB` = ride bell
- `CR` = crash cymbal
- `SD` = snare drum
- `GH` = ghost note snare
- `KK` = kick/bass drum
- `T1` = high tom
- `T2` = mid tom
- `T3` = floor tom
- `RS` = rim shot / cross-stick
- `CP` = clap
- `CB` = cowbell
- `SH` = shaker
- `CL` = clave
- `LD` = log drum
- `TM` = tamborim / timbale
- `CG` = conga (high/slap)
- `CL2` = conga (low/open)
- `AG` = agogo bell

**Velocity Layers:**
- `X` (accent) = velocity 110-127
- `x` (normal hit) = velocity 80-100
- `g` (ghost note) = velocity 25-45
- `o` (open hi-hat) = velocity 75-95

## Swing Implementation
For patterns marked with swing:
- **50%** = straight (no swing)
- **55%** = subtle swing
- **58%** = standard jazz swing (approx. triplet feel)
- **60%** = wide, lazy swing (ballads)
- **67%** = full triplet shuffle

In code, delay the 2nd and 4th 16th notes of every beat by the swing percentage.

## Velocity Guidelines
- `X` (accent) = velocity 110-127
- `x` (normal hit) = velocity 80-100
- `g` (ghost note) = velocity 25-45
- `o` (open hi-hat) = velocity 75-95
