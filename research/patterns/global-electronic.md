# Global Electronic Scenes -- Step-Sequencer Patterns
## Playable Patterns for BeatForge Web Drum Machine

### Notation Key
- Each character = one subdivision position in the cycle
- 16-step patterns = 4/4 time (grouped 4+4+4+4)
- 32-step patterns = two bars of 4/4 (grouped 4+4+4+4|4+4+4+4)
- `x` = hit (normal velocity), `X` = accented hit, `.` = rest
- `G` = ghost note (very low velocity), `o` = open hi-hat, `+` = closed/muted
- Pipe `|` marks beat divisions
- KK = kick, SN = snare, HH = closed hi-hat, OH = open hi-hat
- CL = clap, TM = tom, CB = cowbell, SH = shaker, RS = rimshot
- CG = conga, PR = perc (generic), CR = crash, RD = ride
- 808 = 808 sub-bass, WH = whistle/synth hook, LD = log drum

### Velocity Layers
- `X` (accent) = velocity 110-127
- `x` (normal hit) = velocity 80-100
- `G` (ghost note) = velocity 25-45
- `o` (open hi-hat) = velocity 75-95

### === 2. FUNKOT BASIC (Indonesia) ===
Genre: Funkot / Funky Kota
Origin: Jakarta (Kota district), Indonesia
BPM: 180-190
Time Sig: 4/4
Steps: 16
Difficulty: Intermediate
Description: Indonesia's homegrown hyper-speed house music. The triplet bass kick pattern (inherited from dangdut koplo) distinguishes funkot from European hardcore. Cowbell provides the timeline. Amen break elements and rolling tom fills add texture. Born in Jakarta's Kota nightlife district in the 1990s via Barakatak. 180-220 BPM makes this one of the fastest dance genres globally.

**Basic Pattern (16 steps at 185 BPM):**

```
CB:  |X...|X...|X...|X...|  Cowbell on every beat
HH:  |x.x.x.x.|x.x.x.x.|  16th-note hi-hats
SN:  |....|X...|....|X...|  Snare on 2 and 4
KK:  |X.x.|X.x.|X.x.|X.x.|  Triplet kick pattern (koplo-derived)
TM:  |...x|....|...x|....|  Tom accent before beat 2
```

**Note:** The "triplet kick" is the defining funkot element. At this speed, the KK pattern above approximates the galloping koplo feel. In a true triplet grid, the kicks would fall on beat 1, the triplet "and," then rest.

### === 9. DOMINICAN DEMBOW (Original Riddim) ===
Genre: Dominican Dembow
Origin: Santo Domingo, Dominican Republic
BPM: 115-120
Time Sig: 4/4
Steps: 16
Difficulty: Beginner
Description: The ORIGINAL dembow riddim as developed in the Dominican Republic -- distinct from the Puerto Rican reggaeton version. Pioneered by DJ Boyo (1993, "Mujeres Andadoras"). Faster than reggaeton, stripped to percussive essentials. The "boom-ch-boom-chick" pattern integrated with the 3-2 clave creates a hypnotic, trance-like groove. El Alfa, Rochy RD, Chimbala.

**Dominican Dembow Riddim (16 steps at 118 BPM):**

```
HH:  |x.x.|x.x.|x.x.|x.x.|  Eighth-note hi-hats
CL:  |...x|....|..x.|....|  Clap on tresillo accents
SN:  |....|..X.|....|.X..|  Snare displaced (NOT on 2/4)
KK:  |X...|.X..|X...|.X..|  Kick follows 3+3+2 tresillo
808: |X...|.X..|X...|.X..|  808 doubles the kick
```

**3+3+2 Tresillo Breakdown:**
The kick pattern divides the 8 eighth-notes of a bar as 3+3+2:
- Hit on 1 (count 1)
- Hit on the "and" of 2 (count 4)
- Hit on beat 4 (count 7)
This creates the characteristic "boom...ch-boom...chick" feel.

**Traditional Riddim Pattern (the dotted-note feel):**

```
HH:  |x.x.|x.x.|x.x.|x.x.|  Eighth-note hats
SN:  |..X.|..X.|.X..|..X.|  Dotted-note snare pattern
KK:  |X...|.X..|..X.|....|  Dotted-note kick pattern
```

**Note:** The "dotted-note" shape is dancehall's DNA: groups of **two dotted eighth-notes followed by a regular eighth-note**. This creates a very different feel from the "2-and-4 backbeat" of most Western dance music. The kick and snare trade positions more equally than in other genres.
