# Central Asian, Pacific & Native American Drum Pattern Library
## Playable Step-Sequencer Grid Patterns for Web Drum Machine

### How to Read This Document

Each position in the grid is one **16th note** (unless otherwise noted). A measure of 4/4 = 16 positions.
Odd meters are handled by adjusting the total grid length (e.g., 7/8 = 14 sixteenth notes, 6/8 = 12).

**Central Asian Frame Drum Voices (Doira/Dap):**
- `B` = Bum (bass, center strike) -- deep, resonant
- `T` = Tek/Bak (edge strike) -- bright, high pitch, activates metal rings
- `K` = Ka (muted/ghost) -- soft tap, dead stroke
- `R` = Roll/Riz (finger roll or shake) -- rapid ornamental tremolo
- `F` = Flick/Nouhun (little finger ornament)

**Pacific / Oceanian Voices:**
- `H` = High tone (slit drum near edge / gourd finger tap)
- `L` = Low tone (slit drum center / gourd ground strike)
- `S` = Slap (hand drum open slap)
- `M` = Muted (dampened stroke)

**Native American Voices:**
- `D` = Drum (unified strike on large drum)
- `A` = Accent / Honor beat (hard strike, everyone together)
- `R` = Rattle (gourd rattle or cowhorn rattle)
- `C` = Clap / body percussion

**Kit Adaptation Voices:**
- `KK` = Kick drum (maps from bass/low tones)
- `SN` = Snare / rimshot (maps from tek/high tones)
- `HH` = Hi-hat (maps from ghost notes / ornaments / rattles)

**Symbols:**
- `x` = strike
- `X` = accented strike
- `.` = rest / silence
- `|` = bar line / beat division (visual grouping only)

```
of honor beats is evaluated by judges. Northern style places them
DURING verses; Southern style places them BETWEEN verses.
```

- **Yidaki**: Continuous drone -- not directly possible in step sequencer; represent accents only
